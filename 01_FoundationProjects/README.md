## About This Repository

This folder contains the **Foundation Projects** for the ZAS Robotics Motion (Actuator) Kit.  
Included topics:

- DC Motor Control  
- Servo Motor Basics  
- Stepper Motor Motions  
- TB6612 Motor Driver Usage  
- PWM Speed Control  
- Fundamental Motion Experiments for Robotics

Compatible with:
- Arduino Uno / Nano  
- ZAS Motion Actuator Board  
- Beginner-friendly experiments with step-by-step guidance

Learn motion, build robots, and explore automation!
